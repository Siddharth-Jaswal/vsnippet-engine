# Demo Capture Guide

## Objective

Record short, clean GIFs that demonstrate real VSnippet usage for README and marketplace listing.

## Output Files

Save GIFs in `assets/demo/` with these exact names:

- `01-save-snippet.gif`
- `02-ghost-text.gif`
- `03-tab-commit.gif`
- `04-interference.gif`

## Recording Setup

- Use a clean VS Code profile or Extension Development Host.
- Set zoom to 100% and keep editor font readable (14-16).
- Use a single theme consistently across all GIFs.
- Keep each GIF between 5 and 12 seconds.
- Recommended dimensions: around `1280x720`.

## Storyboard

### 01 Save Snippet

1. Select code block.
2. Run `Save Snippet`.
3. Enter snippet key.
4. Show success message.

### 02 Ghost Text

1. Type snippet prefix.
2. Pause on visible ghost text.

### 03 Tab Commit

1. With ghost text visible, press `Tab`.
2. Show final inserted code.

### 04 Interference Handling

1. Keep other suggestion extension enabled.
2. Trigger VSnippet ghost text.
3. Press `Tab` and show VSnippet commit behavior.

## Quality Bar

- Cursor movement should be intentional and minimal.
- No unrelated windows or notifications.
- Keep loops short and readable.
- Re-record if text is blurry or flow is unclear.
